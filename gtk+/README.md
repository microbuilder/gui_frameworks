# GTK

## OS X

```bash
$ brew install gtk+3
```

At present this will install **GTK+3.24.30**.

- [GTK+3.24.30 Documentation](https://gjs-docs.gnome.org/gtk30/)

For glade GUI editor:

```bash
$ brew install glade
$ glade
```

## Ubuntu 20.04

```bash
$ sudo apt-get install libgtk-3-dev
```
